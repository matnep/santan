# santan

A multiplayer game built with Phaser and Socket.io.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
node server.js
```

3. Open your browser and navigate to `http://localhost:3000`

## Features

- Multiplayer game with Socket.io
- Player customization (skins)
- CRT filter effect (toggleable)

## Controls

- **WASD** or **Arrow Keys**: Move player
- **SPACE**: Open skin menu (when near the house)
